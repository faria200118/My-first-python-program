Python Basics Practice
📌 Project Description

This project is a simple Python practice program created to understand the basic concepts of Python programming.
The script demonstrates variables, data types, arithmetic operators, relational operators, logical operators, assignment operators, comments, and type conversion.

This program is intended for beginner-level Python learners.

📂 Features Demonstrated
1️⃣ Variables and Data Types

The program defines and prints different data types in Python.

Example:

name = "Faria"
age = 25
price = 25.99
old = False
a = None

Data types used:

String

Integer

Float

Boolean

NoneType

The type() function is used to display variable data types.

2️⃣ Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

Operator	Description
+	Addition
-	Subtraction
*	Multiplication
/	Division
%	Modulus
**	Exponent (Power)

Example:

print(b + c)
print(b - c)
print(b * c)
3️⃣ Relational Operators

Relational operators compare two values and return True or False.

Operators used in this program:

== Equal to

!= Not equal

> Greater than

< Less than

>= Greater than or equal

<= Less than or equal

Example:

print(b == c)
print(b != c)
4️⃣ Assignment Operators

Assignment operators are used to update variable values.

Operators demonstrated:

+=

-=

*=

/=

%=

Example:

num = 10
num += 10
5️⃣ Logical Operators

Logical operators combine conditional statements.

Operators used:

and

or

not

Example:

print(not False)
print("AND operator:", val1 and val2)
6️⃣ Type Conversion

The program demonstrates automatic type conversion in Python.

Example:

f = 2
g = 4.25
sum = f + g

Python automatically converts integer to float when performing the addition.

7️⃣ Comments in Python

The program also shows how to write comments.

Single line comment:

# This is a comment

Multi-line comment:

"""
multi-line
comment
"""
▶️ How to Run the Program
Step 1

Install Python on your computer.

Step 2

Save the program file as:

python_basics.py
Step 3

Run the program using the command:

python python_basics.py
